# SEOplugin для Wordpress
Данный ПОЛНОСТЬЮ бесплатный плагин для Wordpress CMS позволяет улучшить сайт в поисковых системах - Google, Yandex, Bing, DuckDuckGo и т.д.

Плагин добавляет:
- Поля для ввода "keywords" и "description"
- Поле для вставки ID счетчика для ЯндексМетрики
- Поле для вставки HTML-кода для ИКС-иконки (веб-мастер Яндекс)

# Как установить плагин
Плагин устанавливается очень просто.
1) Скачиваем архив с [сайта](https://muzharovsky.com);
2) Загружаем его в Wordpress. (В раздел "Плагины");
3) Активируем;
4) Используем его по назначению.
